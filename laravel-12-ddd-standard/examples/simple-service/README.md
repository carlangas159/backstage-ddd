# Simple Service - Ejemplo mínimo

Microservicio ejemplo que demuestra el flujo Controller -> UseCase -> Repository (in-memory) siguiendo las reglas de `laravel-12-ddd-standard`.

Estructura:

```
src/
  Domain/
  Application/
  Infrastructure/
  Web/
```

Este ejemplo está pensado para lectura y como referencia; no requiere ejecutar comandos para su revisión.
